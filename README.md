# 🧪 Proyecto QA Automation - Mercury Tours

Este repositorio contiene pruebas automatizadas desarrolladas en **Java** utilizando **Selenium WebDriver** y gestionadas mediante **Maven** para el sitio web Mercury Tours.

## 🛠️ Tecnologías
* JDK 21
* Selenium WebDriver
* JUnit 5 / TestNG
* Maven
